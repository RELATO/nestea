# NESTEA - Node, Express4, Sequelize, TypeScript, Epilogue and Angular

Requirements
------------
Have Node.js and npm installed.

How to run
----------
```
git clone https://github.com/RELATO/nestea
npm install -g gulp
npm install -g typescript
npm install -g typings
npm install
typings install
gulp serve
```
